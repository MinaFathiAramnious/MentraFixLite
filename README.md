**MentraFix Lite** هو تطبيق ويب محلي بالكامل (Local Web App) مصمم لمساعدة مهندسي الصيانة وأصحاب المراكز في إدارة دورة العمل بالكامل (عملاء، أجهزة، أوامر شغل، قطع غيار، فواتير، ومصروفات) بخصوصية تامة وبدون الحاجة للاتصال بالإنترنت أو دفع اشتراكات شهرية.

---

## ✨ المميزات الرئيسية (Core Features)

- 🔒 **خصوصية تامة (Offline First):** تعمل قاعدة البيانات بالكامل داخل متصفح المستخدم (IndexedDB) عبر مكتبة `Dexie.js`. لا توجد خوادم (Backend) ولا يتم رفع أي بيانات للسحابة.
- 👥 **إدارة العملاء والأجهزة:** تسجيل بيانات العملاء وربط كل عميل بأجهزته الخاصة (النوع، الماركة، الموديل، السيريال).
- ⚙️ **ورشة الصيانة (أوامر الشغل):** إنشاء "تيكت صيانة"، تتبع حالة الأجهزة (قيد الفحص، جاري الصيانة، بانتظار قطع غيار، مكتمل)، وتسجيل الشكوى والملاحظات.
- 📦 **المخزن والمشتريات:** إدارة قطع الغيار، إدخال فواتير المشتريات، وخصم قطع الغيار المستخدمة في الصيانة من المخزن تلقائياً.
- 💰 **المالية والفواتير:** إصدار فواتير للعملاء، تحصيل المبالغ بطرق دفع مختلفة، تسجيل المصروفات التشغيلية، وحساب صافي الربح لحظياً.
- 💾 **نظام النسخ الاحتياطي (Backup):** أداة مدمجة لتصدير قاعدة البيانات بالكامل في ملف، واسترجاعها في أي وقت لحماية البيانات من الضياع.
- 🚀 **متعدد المراكز (Multi-tenant):** إمكانية تأسيس أكثر من مركز صيانة على نفس الجهاز، ولكل مركز قاعدة بيانات وكلمة مرور مستقلة تماماً.
- 📱 **تطبيق ويب تقدمي (PWA):** يمكن تثبيت النظام كبرنامج على أجهزة الكمبيوتر (Windows/Mac) أو الهواتف الذكية للعمل كبرنامج مستقل (Standalone).

## 🛠️ التقنيات المستخدمة (Tech Stack)

تم بناء هذا المشروع بهيكلية ذكية جداً ليعمل في المتصفح مباشرة دون الحاجة لـ Build Tools معقدة:
- **الواجهة الأمامية:** `React.js` (عبر CDN).
- **مترجم الكود (Compiler):** `Babel Standalone` لترجمة أكواد الـ JSX داخل المتصفح في وقت التشغيل (Runtime).
- **التصميم:** `Tailwind CSS` (عبر CDN) مع أيقونات `FontAwesome` وخطوط `Google Fonts (Cairo)`.
- **قاعدة البيانات:** `Dexie.js` (لإدارة قواعد بيانات الـ IndexedDB بمرونة عالية).
- **التوجيه (Routing):** نظام تحميل ديناميكي للشاشات عبر جلب ملفات الـ `.js` من مجلد `pages/` وحقنها في الصفحة.
